# 🗓 올인원 라이브 코딩 테스트 대비 계획표 (수요일 시작 기준)

**기본 사이클**: `직접 구현(Self)` -> `AI 면접관 시점 리뷰(Review)` -> `피드백 반영(Refactor)` -> `망각 곡선 극복을 위한 1차 복습(Revision)`
**목표**: 다음 주 수요일 아침 면접을 대비하여 **화요일까지 모든 훈련 종료**. (Spring Boot + 객체지향 + PostgreSQL 올인원)

## 📌 Day 1 (수): 워밍업 & 환경 세팅
- [x] **환경 세팅**: 제공된 `docker-compose.yml`을 실행하여 로컬 PostgreSQL DB 띄우기
- [x] **순수 자바 1**: `parking` 패키지에 주차장 로직 구현 (객체 상태 관리, Map/Set 활용)
- [x] **리뷰 및 복습**: AI 면접관에게 코드 리뷰 요청 후 피드백 반영

## 📌 Day 2 (목 / 오늘): SQL 기초 & 동시성 제어 돌입
- [ ] **SQL 1**: `sql` 디렉토리에 기초 테이블(User, Order) 생성 DDL 작성 및 단순 JOIN, GROUP BY 쿼리 연습
- [ ] **Spring 1**: 재고 차감 동시성 제어 (비관적 락 vs 낙관적 락 실제 적용)
- [ ] **리뷰 및 1차 복습**: 작성된 쿼리와 락 코드 리뷰 받기 + **[복습]** 전날 진행한 `parking` 로직 눈으로 훑으며 리마인드

## 📌 Day 3 (금): 캐시(Redis) & 복잡한 SQL
- [ ] **Spring 2**: 쿠폰 선착순 발급 로직 (Redis Lua Script 등 활용)
- [ ] **SQL 2**: 서브쿼리와 인라인 뷰를 활용한 다중 테이블 집계 쿼리 연습
- [ ] **리뷰 및 1차 복습**: Redis 정합성 관련 코드 리뷰 + **[복습]** Day 2에 진행한 `재고 차감 락(Lock)` 코드 백지에서 핵심 로직만 손코딩(또는 타이핑) 해보기

## 📌 Day 4 (토): 결제 아키텍처 & 고급 SQL
- [ ] **Spring 3**: 외부 PG 연동 시나리오 기반 멱등성(Idempotency) 방어 로직 구현
- [ ] **SQL 3**: 윈도우 함수(`SUM OVER`, `PARTITION BY`)를 활용한 누적합, 이동평균 쿼리 작성
- [ ] **리뷰 및 1차 복습**: 외부 API 통신 트랜잭션 분리 리뷰 + **[복습]** Day 3의 `Redis Lua Script` 로직 및 `서브쿼리` 작성법 리마인드

## 📌 Day 5 (일): 고급 쿼리 마무리 & 전체 흐름 복습
- [ ] **SQL 4**: `WITH RECURSIVE`를 활용한 계층형 쿼리(카테고리 트리) 작성 연습
- [ ] **1주기 종합 복습**: 지금까지 작성한 Spring Boot 코드(동시성, Redis, 결제)와 SQL 코드를 처음부터 끝까지 읽으며, "왜 이 기술을 도입했는가?"에 대한 스스로의 답변 정리

## 📌 Day 6 (월): 실전 모의고사
- [ ] **타임어택**: 지금까지 한 문제 중 1개를 랜덤 지정받아 40분 안에 아무것도 없는 상태에서 구현
- [ ] **SQL 타임어택**: AI 면접관이 즉석에서 내는 비즈니스 요구사항을 SQL 쿼리로 단번에 작성
