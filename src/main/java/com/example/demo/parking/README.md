# 🚗 주차장 시스템 (순수 자바 OOP)

## 📌 개요
프레임워크(Spring) 없이 순수 자바의 객체지향적 설계와 자료구조 활용 능력을 평가합니다.

## 🎯 요구사항
1. **차량 입차 (Park)**
   - 주차장은 최대 수용 가능 대수(`capacity`)를 가집니다.
   - 차량의 번호판을 받아 빈 자리가 있으면 주차권(`Ticket`)을 발급합니다.
   - 주차권에는 '차량 번호'와 '입차 시간'이 기록됩니다.
   - 만약 만차 상태라면 `IllegalArgumentException` 등 적절한 예외를 발생시킵니다.
2. **차량 출차 (Leave)**
   - 주차권과 출차 시간을 입력받아 해당 차량을 주차장에서 빼냅니다.
   - 주차 시간(출차 시간 - 입차 시간)을 계산하여 요금을 부과합니다. (기본: 시간당 1,000원, 올림 처리)
   - 존재하지 않는 주차권일 경우 예외를 발생시킵니다.

## 💡 평가 포인트 (리뷰 기준)
- `Map`이나 `Set` 등 적절한 컬렉션을 사용하여 `O(1)` 혹은 효율적인 시간 복잡도로 차량을 찾을 수 있는가?
- 상태를 가지는 객체(Ticket, ParkingLot 등)의 캡슐화가 잘 되어 있는가?
- 테스트 코드 작성을 고려하여 `LocalDateTime` 같은 시간 객체를 외부에서 주입받도록 설계했는가?
