# 🎰 로또 시스템 (순수 자바 TDD)

## 📌 개요
비즈니스 룰을 객체 내부에 안전하게 캡슐화하고, 테스트하기 어려운 로직(Random)을 분리하는 역량을 평가합니다.

## 🎯 요구사항
1. **로또 구매**
   - 구입 금액을 입력하면 1,000원 단위로 로또를 자동 발급합니다.
   - 각 로또는 1~45 사이의 중복되지 않는 6개의 숫자를 가집니다.
2. **당첨 확인 및 통계**
   - 지난 주 당첨 번호(6개)와 보너스 번호(1개)를 입력받습니다.
   - 발행된 로또들과 비교하여 일치하는 번호 개수(3개~6개)를 구합니다.
   - 총 당첨 금액을 합산하고, 구매 금액 대비 **총 수익률**을 계산하여 반환합니다.

## 💡 평가 포인트 (리뷰 기준)
- 로또 번호(`LottoNumber`), 로또 1장(`Lotto`)을 **불변 객체**이자 **일급 컬렉션**으로 포장했는가?
- 무작위 번호를 추출하는 `Random` 로직을 인터페이스로 추상화하여, 단위 테스트 시 원하는 번호를 고정으로 뽑을 수 있도록(Mock/Fake) 제어했는가?
