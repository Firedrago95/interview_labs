# 🗓 올인원 라이브 코딩 테스트 대비 계획표 (수요일 시작 기준)

**기본 사이클**: `직접 구현(Self)` -> `AI 면접관 시점 리뷰(Review)` -> `피드백 반영(Refactor)`
**목표**: 다음 주 수요일 아침 면접을 대비하여 **화요일까지 모든 훈련 종료**. (Spring Boot + 객체지향 + PostgreSQL 올인원)

## 📌 Day 1 (수 / 오늘): 워밍업 & 환경 세팅
- [ ] **환경 세팅**: 제공된 `docker-compose.yml`을 실행하여 로컬 PostgreSQL DB 띄우기
- [ ] **순수 자바 1**: `parking` 패키지에 주차장 로직 구현 (객체 상태 관리, Map/Set 활용)
- [ ] **리뷰 및 복습**: AI 면접관에게 코드 리뷰 요청 후 피드백 반영

## 📌 Day 2 (목): 도메인 설계 & SQL 기초
- [ ] **순수 자바 2**: `lotto` 패키지에 로또 도메인 구현 (불변 객체 유지, Random 인터페이스 분리 테스트)
- [ ] **SQL 1**: `sql` 디렉토리에 기초 테이블(User, Order) 생성 DDL 작성 및 단순 JOIN, GROUP BY 쿼리 연습
- [ ] **리뷰 및 복습**: 로또 TDD 리뷰 및 SQL 쿼리 최적화 리뷰

## 📌 Day 3 (금): Spring Boot 동시성 제어 & SQL 서브쿼리
- [ ] **Spring 1**: 재고 차감 동시성 제어 (비관적 락 vs 낙관적 락 실제 적용)
- [ ] **SQL 2**: 서브쿼리와 인라인 뷰를 활용한 다중 테이블 집계 쿼리 연습
- [ ] **리뷰 및 복습**: 락 적용 시 데드락 위험성 리뷰, SQL 실행 계획(EXPLAIN) 리뷰

## 📌 Day 4 (토): Spring Boot 성능/분산 & 윈도우 함수
- [ ] **Spring 2**: 쿠폰 선착순 발급 로직 (Redis Lua Script 등 활용)
- [ ] **SQL 3**: 윈도우 함수(`SUM OVER`, `PARTITION BY`)를 활용한 누적합, 이동평균 쿼리 작성
- [ ] **리뷰 및 복습**: 단일 장애점 방어 리뷰, 윈도우 함수 로직 피드백

## 📌 Day 5 (일): 결제 아키텍처 & 고급 SQL
- [ ] **Spring 3**: 외부 PG 연동 시나리오 기반 멱등성(Idempotency) 방어 로직 구현
- [ ] **SQL 4**: `WITH RECURSIVE`를 활용한 계층형 쿼리(카테고리 트리) 작성 연습
- [ ] **리뷰 및 복습**: 트랜잭션 경계 설정 리뷰, 재귀 쿼리 무한루프 방지 피드백

## 📌 Day 6 (월): 실전 모의고사
- [ ] **타임어택**: 지금까지 한 문제 중 1개를 랜덤 지정받아 40분 안에 아무것도 없는 상태에서 구현
- [ ] **SQL 타임어택**: AI 면접관이 즉석에서 내는 비즈니스 요구사항을 SQL 쿼리로 단번에 작성
- [ ] **약점 보완**: 타임어택에서 막혔던 부분 집중 복습

## 📌 Day 7 (화): 멘탈 관리 및 압박 면접
- [ ] **기술 인터뷰**: 구현한 코드를 바탕으로 "트래픽이 100배 증가하면?", "DB 커넥션이 고갈되면?" 등의 꼬리 질문 방어 연습
- [ ] **최종 마인드 컨트롤**: 모든 구현 코드를 눈으로 읽으며 구조를 체화 (이후 휴식)
