# 📝 리뷰 및 실시간 평점 집계 시스템

## 📌 개요
데이터의 잦은 쓰기(리뷰 작성)와 잦은 읽기(평점 조회)가 혼재하는 도메인에서 조회 성능을 높이기 위한 역량을 평가합니다.

## 🎯 요구사항
1. **리뷰 작성**
   - 구매한 유저만 해당 상품에 대해 1~5점 사이의 평점과 리뷰 내용을 남길 수 있습니다.
2. **상품 평점 집계**
   - 리뷰가 작성될 때마다 상품의 '평균 평점'과 '리뷰 총 개수'가 실시간으로 반영되어야 합니다.
   - 수백만 건의 리뷰가 있을 때, 매번 `AVG()` 쿼리를 날리면 시스템이 멈춥니다.

## 💡 평가 포인트 (리뷰 기준)
- 리뷰가 생성/수정/삭제될 때, 상품별 평점 요약 테이블(`product_rating_summary`)을 어떻게 원자적으로 업데이트할 것인가?
- 동시 작성 시 평점 데이터의 정합성(Race condition)을 어떻게 보장할 것인가?
